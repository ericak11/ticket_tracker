# Tix Tracker

Tix Tracker is an app that tracks the sale of season tickets. It uses the SeatGeek API to find home team game times and dates.

Sign up for [Tix Tracker](http://tixtracker.herokuapp.com/)
.
## Technologies Used

* Ruby - 2.3.0
* Rails - 5.0.0
* Bootstrap
* Httparty - 0.13.7

## Future Features
